# Phase 6: UI Refinement

## Objective
Polish the UI to match Blinkit's visual identity — vibrant colors, clean typography, responsive design, and micro-animations.

---

## Scope
- Blinkit-inspired color scheme (yellow/green primary)
- Typography refinement (Inter/Outfit font)
- Responsive grid layout
- Micro-animations (hover, add-to-cart, recommendation appear)
- Navbar polish (logo, cart counter badge)

---

## Tasks

### 6.1 Design System
- Define Tailwind color palette matching Blinkit (yellow #F8CB46, green #0C831F)
- Configure custom fonts (Inter via Google Fonts)
- Define spacing, border-radius, shadow tokens

### 6.2 Navbar Refinement
- Blinkit-style navbar with logo, search bar placeholder, cart icon
- Cart counter badge with animation on increment
- Sticky navbar on scroll

### 6.3 Product Card Polish
- Card hover effects (subtle lift + shadow)
- "Add" button with green accent color
- Price formatting (₹ symbol)
- Image placeholder with gradient background

### 6.4 Recommendation Section Polish
- Slide-in animation when recommendations load
- Score displayed as percentage badge
- "Frequently Bought Together" header with icon
- Horizontal scroll with snap points

### 6.5 Responsive Design
- Mobile: 2-column grid
- Tablet: 3-column grid
- Desktop: 4-5 column grid
- Recommendation section scrollable on all sizes

---

## Expected Output
- Polished, Blinkit-inspired UI
- Smooth interactions and transitions
- Fully responsive layout

---

## Acceptance Criteria
- [ ] Color scheme matches Blinkit palette
- [ ] Custom font loaded and applied
- [ ] Hover animations on product cards
- [ ] Cart badge animates on add
- [ ] Recommendation section slides in
- [ ] Responsive on mobile/tablet/desktop

---

## Out of Scope
- Dark mode
- Accessibility audit (a11y)
- SEO optimization (basic meta tags only)
- Internationalization
- Custom illustrations/icons
